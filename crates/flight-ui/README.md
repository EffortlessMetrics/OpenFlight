# flight-ui

UI-facing integration types and settings for OpenFlight.

## Responsibilities

- Defines UI integration helpers and settings-related types.
- Acts as a boundary crate between UI code and service/core APIs.
- Contains documentation-oriented integration stubs used by the workspace.

## Key Modules

- `src/integration_docs.rs`
- `src/settings.rs`

## Status

Internal workspace crate; APIs can evolve with Flight Hub service requirements.
