# flight-test-helpers

Reusable helpers for OpenFlight unit and integration tests.

## Responsibilities

- Shared test fixtures/builders.
- Shared assertions for adapter state and bus snapshot validity.
- Polling/synchronization helpers for timing-sensitive tests.
- Lightweight integration test harness utilities.

## Key Modules

- `src/fixtures.rs`
- `src/assertions.rs`
- `src/utils.rs`
- `src/integration.rs`
