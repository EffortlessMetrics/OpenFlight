# flight-panels-cougar

Thrustmaster Cougar MFD panel driver implementation.

## Responsibilities

- Implements Cougar MFD HID mapping and state conversion.
- Integrates Cougar panel behavior with panel-core abstractions.
- Publishes panel telemetry and health metrics for service consumption.
